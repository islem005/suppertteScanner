# Colors — CSS Token Palette

All color values live in `css/tokens.css` as CSS custom properties. Never hardcode hex values.

## Backgrounds — Warm Dark Slate

| Token | Value | Usage |
|---|---|---|
| `--bg-base` | `#0c0c0d` | Page background |
| `--bg-surface` | `#18181b` | Card backgrounds, table wrappers |
| `--bg-elevated` | `#1f1f23` | Modals, dropdowns, elevated panels |
| `--bg-hover` | `#27272a` | Hover states, icon button backgrounds |
| `--bg-inset` | `#101012` | Inputs, code blocks, inset areas |

## Borders

| Token | Value | Usage |
|---|---|---|
| `--border-subtle` | `rgba(255,255,255,0.06)` | Card borders, table row dividers |
| `--border-default` | `rgba(255,255,255,0.10)` | Input borders, active borders |
| `--border-strong` | `rgba(255,255,255,0.15)` | Focused or emphasized borders |

## Text

| Token | Value | Usage |
|---|---|---|
| `--text-primary` | `#fafafa` | Headings, body copy |
| `--text-secondary` | `#a1a1aa` | Labels, secondary info, metatext |
| `--text-tertiary` | `#71717a` | Captions, timestamps, placeholders |
| `--text-disabled` | `#52525b` | Disabled states |

| Token | Value | Usage |
